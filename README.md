# Algorithm Visualizer & Benchmarker

Проєкт для демонстрації роботи алгоритмів сортування та структур даних з вимірюванням їхньої продуктивності. Програма складається з незалежної бібліотеки алгоритмів та GUI-застосунку для їх візуалізації.

## Функціонал
* Візуалізація роботи алгоритмів сортування у реальному часі.
* Збір метрик (час виконання, кількість ітерацій).
* Інтерактивний режим: крок вперед, крок назад, пауза.

## Архітектура та Патерни Проєктування (GoF)
Проєкт спроєктовано з використанням об'єктно-орієнтованого підходу. Для досягнення максимальної гнучкості реалізовано наступні 12 патернів:

**Породжувальні (Creational):**
1.  **Singleton** — менеджер конфігурацій (тема інтерфейсу, швидкість анімації).
2.  **Factory Method** — генерація різних наборів вхідних даних.
3.  **Builder** — покрокова побудова складних тестових сценаріїв.

**Структурні (Structural):**
4.  **Facade** — єдиний інтерфейс взаємодії між GUI та бекендом алгоритмів.
5.  **Decorator** — обгортки для вимірювання часу виконання функцій.
6.  **Adapter** — інтеграція вбудованої функції `list.sort()` у загальний інтерфейс.
7.  **Composite** — робота зі складними ієрархічними структурами даних.

**Поведінкові (Behavioral):**
8.  **Strategy** — динамічна підміна алгоритмів сортування.
9.  **Template Method** — базовий каркас виконання алгоритму.
10. **Command** — інкапсуляція дій користувача (запуск, пауза) в об'єкти.
11. **Memento** — збереження знімків стану масиву для функції "крок назад".
12. **Iterator** — уніфікований обхід структур даних.
